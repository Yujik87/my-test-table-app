Requirement: installed nodeJS, yarn, WebStorm(for example) on your PC.

1. Download project from gitHub (press button "Code" and choose "GitHub CLI" where you can copy command for your terminal or download archive below)
2. Download modules for this project by command in terminal: yarn
3. When modules are installed and dependencies are created you can start the project by command in terminal: yarn start
4. IDE start server on localhost:3000
